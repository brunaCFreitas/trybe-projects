# Projeto Pixels Art

## Descrição do Projeto

Neste projeto foi implementado um editor de arte com pixels onde a pessoa usuária poderia escolher uma **cor** em uma paleta de cores que deve ser gerada **aleatóriamente** a cada clique no botão `cores aleatorias`. Além disso, este poderá escolher um tamanho de quadro que deve variar entre 5 e 50, sendo sua dimensão n X n.
